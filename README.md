# Template rapport PatriNat

Le dossier permet de produire un rapport en Rmarkdown selon la mise en forme attendue par PatriNat.

Le dossier "couverture" comprends les documents *couverture.docx* et *couverture_dos.docx* qu'il faut:
1) modifier selon les besoins puis,
2) enregistrer au format pdf sans modifier les noms et le répertoire de sortie.
